# 🔗 Hotkey Chain（Chrome 扩展）

<div align="right"><a href="./README.md">English</a> | 简体中文</div>

> **浏览器里的快捷指令。** 把几十个浏览器动作编排成一条"动作链"，再用你喜欢的任意方式触发——快捷键、地址栏、右键、定时，或在页面网址匹配规则时自动运行。

就像 macOS / iOS 的「快捷指令」，但活在 Chrome 里。动作序列只需配置一次，之后随处触发，并支持逐步延迟、条件判断、变量，以及链调用链。

## ✨ 核心亮点

- **75+ 动作**，覆盖页面、标签、窗口、媒体、内容，以及浏览器本身
- **五类触发方式**——快捷键、图标、右键、地址栏、定时、网址自动运行
- **流程控制**：条件、子链、模板变量，让一条链像一段小脚本
- **可视化编辑器**：分组动作选择、逐步延迟、拖拽排序
- **模板库**：一键添加现成功能链（专注模式、标签大扫除、视频模式……）
- **备份与分享**：整份配置导出/导入，或把单条链导出为独立文件
- **18 种语言**：整套界面可在 English、简体中文、日本語、العربية 等 18 种语言间切换

## ⚡ 触发方式

| 触发方式 | 说明 |
| --- | --- |
| **快捷键** | `Ctrl+Shift+H` 运行默认链；`Ctrl+Shift+1/2/3` 运行链 1–3；链 4–9 提供可绑定槽位 |
| **工具栏图标** | 点击运行默认链；右键弹出你的动作链菜单 |
| **页面右键菜单** | 在页面、选中文字、链接、图片或媒体上直接运行链 |
| **地址栏** | 输入 `hc` + 空格 + 链名称即可查找并运行（omnibox） |
| **定时** | 每 N 分钟在后台运行一条链（`chrome.alarms`） |
| **网址自动运行** | 页面加载完成且网址匹配规则时自动运行（带防循环冷却） |

## 🎬 支持的动作

**页面** — 回到顶部/底部、上下滚动一屏、刷新、前进/后退、全屏、深色模式、翻译页面、打印、打开网址（新/当前标签页，支持变量）

**标签** — 新建 / 关闭 / 关闭其他 / 关闭左侧或右侧 / 关闭重复 · 按网址排序 · 按网站分组与取消分组 · 复制 / 固定 / 静音（单个或全部）· 全部刷新 · 休眠其他标签释放内存 · 跳到发声标签 · 收藏所有标签 · 左移 / 右移 / 移到最左 / 最右 · 上一个 / 下一个 · 重开已关闭标签

**窗口** — 新建 / 关闭 / 最小化 / 最大化 · 打开无痕窗口 · 标签移到新窗口

**媒体** — 播放/暂停 · 加速 / 减速 / 重置播放速度 · 朗读选中文字（TTS）· 停止朗读

**内容** — 复制网址 / 标题 / 选中文字 · 复制为 Markdown 链接 · 用默认搜索引擎搜索选中文字 · 加入书签（自动去重）· 加入阅读清单

**缩放** — 放大 / 缩小 / 重置

**高级** — 硬刷新 · 截图 · 保存页面为 MHTML · 系统通知 · 打开浏览器内置页面（下载、历史、设置……）· 打开下载文件夹 · 清除浏览器缓存 · 清除本站数据 · 从历史删除本页 · 保持唤醒开关

**扩展管理** — 启用/禁用、卸载（需确认）、重新加载开发扩展、启动 Chrome 应用、查看扩展信息、打开其选项/主页/商店页，或执行 Chrome 命令（含本扩展自己的动作链）

## 🔀 流程控制与变量

一条链不只是固定列表，还能分支与组合：

- **条件** — 「网址匹配才继续」「有选中文字才继续」，条件不满足时提前停止本链
- **子链** — 「运行另一条链」把链作为一个步骤调用（带防循环与嵌套深度限制）
- **变量** — 在「打开网址」「显示通知」文本中使用 `{url}` `{title}` `{selection}` `{clipboard}` `{date}` `{time}`（例如在任意网站搜索选中文字）
- **运行反馈** — 链运行时工具栏显示徽章；失败时弹出错误徽章 + 系统通知

## 📦 安装

1. 克隆或下载本仓库
2. 打开 `chrome://extensions` 并启用**开发者模式**
3. 点击**加载已解压的扩展程序**，选择 `hotkey-chain` 文件夹
4. 工具栏出现 🔗 图标

在 `chrome://extensions/shortcuts` 自定义快捷键（选项页工具栏的键盘图标可直达）。

### 打包构建

生成一个干净的分发包（只含扩展运行所需文件，排除 `design/`、`docs/`、`scripts/`）：

```bash
npm run package    # 或：node scripts/package.mjs
```

产物（在 `dist/`，已 git 忽略）：

- `dist/hotkey-chain/` —— 解压目录，可直接「加载已解压的扩展程序」
- `dist/hotkey-chain-v<版本>.zip` —— 可上传 Chrome 网上应用店（manifest 位于压缩包根目录）

无需任何依赖；脚本仅用 Node 内置能力 + 系统 zip 工具。

## 🚀 快速上手

1. 点击 🔗 图标运行默认链，或右键打开动作链菜单
2. 打开**选项**管理动作链——从模板开始，或自己搭建
3. 在链中：添加动作、设置逐步延迟、拖拽排序，再绑定快捷键或触发器
4. 点击**测试执行**在当前标签页试运行

## 🛠️ 使用与配置

- 选项页是可视化编辑器：分组动作选择、逐步延迟（毫秒）、拖拽排序
- 每条链可携带自己的触发器——定时间隔与网址自动运行规则
- **执行命令** 列出任意扩展（含本扩展）的命令；**调用扩展** 发送结构化消息（模板或自定义 JSON）
- 动作链与顺序存储在 `chrome.storage.local`
- 工具栏的**导出/导入**按钮可备份或恢复整份配置；编辑器中的**导出此链**把单条链分享为文件

## 🌍 国际化

- **18 种语言**：English、简体中文、繁體中文、日本語、한국어、Español、Français、Deutsch、Português (Brasil)、Русский、Italiano、العربية、हिन्दी、Bahasa Indonesia、Türkçe、Tiếng Việt、ไทย、Polski——默认跟随浏览器
- 选项页提供语言选择器，覆盖整套扩展——后台、菜单、通知、命令；阿拉伯语自动启用从右到左（RTL）布局
- 语言文件位于 `_locales/<code>/messages.json`（如 `en`、`zh_CN`、`ja`、`ar`）
