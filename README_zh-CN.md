# 🔗 Hotkey Chain（Chrome 扩展）

<div align="right"><a href="./README.md">English</a> | 简体中文</div>

> 一个提高效率的 Chrome 扩展：用一个快捷键执行一串动作链。可视化配置多个自定义动作序列，集成常用的扩展管理能力。

## ✨ 功能特性

核心

- 一键执行预设的动作链
- 支持多个动作链，拖拽排序
- 点击工具栏图标执行默认链
- 每个动作可设置延迟（毫秒）

扩展管理

- 启用/禁用扩展
- 卸载扩展（需要确认）
- 重新加载开发中的扩展
- 启动 Chrome 应用
- 显示扩展详情（模态/通知）
- 快速跳转：选项 / 主页 / 商店页面

支持的动作

- 页面：回到顶部/底部、刷新、前进/后退、全屏切换
- 标签：关闭/新建/复制/固定
- 内容：复制 URL/标题、加入书签、聚焦地址栏
- 缩放：放大/缩小/重置
- 高级：清缓存、等待、调用扩展、执行命令

内置示例

- 阅读模式、标签工具、快速收藏、开发强刷
- 命令演示：展示如何执行命令（含自身扩展）

## 📦 安装

加载已解压的扩展

1) 克隆或下载本仓库
2) 打开 chrome://extensions 并启用“开发者模式”
3) 选择“加载已解压的扩展程序”，指向 hotkey-chain 文件夹
4) 工具栏会出现 🔗 图标

键盘快捷键（可在 chrome://extensions/shortcuts 自定义）

- Ctrl+Shift+H：运行默认动作链
- Ctrl+Shift+1/2/3：运行动作链 1/2/3

图标右键菜单

- 运行默认动作链
- 运行动作链 1/2/3
- 打开设置（选项页）

## 🚀 快速上手

- 点击图标运行默认链
- 右键图标打开快速菜单
- 打开“选项”编辑动作链：新增动作、设置延迟、拖拽排序

## 🛠️ 使用与配置

- 选项页提供可视化编辑器
- “执行命令”会列出某个扩展（包含本扩展）的可用命令
- “调用扩展”支持通过模板或自定义 JSON 发送消息
- 动作链与顺序存储在 chrome.storage.sync

## 🧩 动作与命令

动作类别覆盖页面/标签/内容/缩放/高级/扩展。你可以：

- 执行 Chrome 命令（例如运行本扩展的 chain_1）
- 切换其他扩展的启用状态
- 打开其他扩展的选项/详情/主页
- 获取扩展详细信息（权限、安装类型、版本等）

## 🌍 国际化（i18n）

- 默认语言：英语（en）。可选：en、zh_CN
- 选项页提供语言选择器；你的选择会覆盖整套扩展（后台、菜单、通知、命令）
- 语言文件位于 `_locales/en/messages.json` 与 `_locales/zh_CN/messages.json`
